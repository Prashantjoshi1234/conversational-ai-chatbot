#  Conversation AI Chatbot (RAG + ChromaDB)

An intelligent **Conversational AI Chatbot** built using Retrieval-Augmented Generation (RAG).  
It uses **ChromaDB** as a vector database to retrieve relevant context from documents and generate accurate responses using an LLM.

Generates responses based on conversation history for contextual understanding.

---

##  Features
-  Conversational AI chatbot
-  Document-based knowledge system (RAG)
-  Semantic search using embeddings
-  Top-K relevant document retrieval
-  Fast vector search with ChromaDB
-  Secure API key management using `.env`
-  Knowledge base creation using multiple sources like web crawling, PDFs

---

##  Tech Stack
- Python 
- Streamlit (Frontend UI)
- LLaMA (Large Language Model)
- ChromaDB (Vector Database)
- Sentence Transformers / Embedding Model
- FastAPI (optional)
- python-dotenv
